Identity type

In HoTT, we have a different interpretation of type theory in which the set-theoretical notion of sets for types is replaced by the topological notion of spaces.

{-# OPTIONS --without-K #-}
open import BasicTypes     public
open import BasicFunctions public

Homogeneous equality

The equality type also called identity type is considered a primary type (included in the theory by default). To form this type, we consider a type A which the terms of the equalities are. In addition, we fix a term a : A and the identity type is a == a also denoted by Id(a,a) or a↝a. We count with only one constructor for its terms which is idp also called in other sources as refl. Their names stands for the reflexive paths. To work with identity types, we have below the J-eliminator.

data
  _==_ {A : Type ℓᵢ} (a : A)
    : A → Type ℓᵢ
  where
  idp : a == a

-- synonyms for the identity type
Eq   = _==_
Id   = _==_
Path = _==_
_⇝_  = _==_   -- '\r~'
_≡_  = _==_   -- '\equiv'

infix 30 _==_ _⇝_ _≡_

{-# BUILTIN EQUALITY _==_  #-}

refl
  : ∀ {A : Type ℓᵢ}
  → (a : A)
  -----------------
  → a == a

refl {ℓᵢ}{A} a = idp

Symmetry property for the identity types.

sym
  : ∀ {A : Type ℓ}{x y : A}
  → x == y
  ----------------------------
  → y == x

sym idp = idp

syntax sym p = − p

J eliminator

To work with identity types, we have its induction principle as the J-eliminator.

Paulin-Mohring J rule

J
  : ∀ {A : Type ℓᵢ} {a : A}
  → (B : (a' : A) (p : a == a') → Type ℓⱼ)
  → (d : B a idp)
  ----------------------------------------
  → {a' : A} (p : a == a') → B a' p

J {a = a} B d idp = d

J'
  : ∀ {A : Type ℓᵢ} {a : A}
  → (B : (a' : A) (p : a' == a) → Type ℓⱼ)
  → (d : B a idp)
  ----------------------------------------
  → {a' : A} (p : a' == a) → B a' p

J' {a = a} B d idp = d

Composition of paths

_·_
  : ∀ {A : Type ℓ} {x y z : A}
  → (p : x == y)
  → (q : y == z)
  --------------
  → x == z

_·_ idp q = q

infixl 50 _·_

Inverse of paths

inv
  : ∀ {ℓ} {A : Type ℓ} {a b : A}
  → a == b
  --------
  → b == a

inv idp = idp

-- synonyms for inverse path
_⁻¹ = inv
!_  = inv

infixl 60 _⁻¹ !_

Associativity of composition

  • Left associativity
∘-lassoc
  : ∀ {ℓ} {A B C D : Type ℓ}
  → (h : C → D) → (g : B → C) → (f : A → B)
  -----------------------------------------
  → (h ∘ (g ∘ f)) == ((h ∘ g) ∘ f)

∘-lassoc h g f = idp {a = (λ x → h (g (f x)))}
  • Right associativity
∘-rassoc
  : ∀ {ℓ} {A B C D : Type ℓ}
  → (h : C → D) → (g : B → C) → (f : A → B)
  -----------------------------------------
  → ((h ∘ g) ∘ f) == (h ∘ (g ∘ f))

∘-rassoc h g f = (∘-lassoc h g f) ⁻¹

Heterogeneous equality

data
  HEq {ℓ} (A : Type ℓ)
    : (B : Type ℓ)
    → (α : A == B) (a : A) (b : B)
    → Type (lsuc ℓ)
  where
  idp : ∀ {a : A} → HEq A A idp a a

Equational reasoning

Equational reasoning is a way to write readable chains of equalities.

  t : a == e
  t = a =⟨ p ⟩
      b =⟨ q ⟩
      c =⟨ r ⟩
      d =⟨ s ⟩
      e ∎

where p is a path from a to b, q is a path from b to c, and so on.

module
  EquationalReasoning
  {A : Type ℓᵢ}
  where

Definitional equals:

  -- Reasoning.
  _==⟨⟩_
    : ∀ (x {y} : A)
    → x == y → x == y

  _ ==⟨⟩ p = p

  -- synonyms for _==⟨⟩
  _==⟨idp⟩_  = _==⟨⟩_
  _==⟨refl⟩_ = _==⟨⟩_
  _≡⟨⟩_      = _==⟨⟩_

  infixr 2 _==⟨⟩_ _==⟨idp⟩_ _==⟨refl⟩_ _≡⟨⟩_

  -- chain
  _==⟨_⟩_
    : (x : A) {y z : A}
    → x == y
    → y == z
    → x == z

  _ ==⟨ thm ⟩ q = thm · q

  _≡⟨_⟩_ = _==⟨_⟩_
  infixr 2 _==⟨_⟩_ _≡⟨_⟩_

  -- Q.E.D
  infix 3 _∎
  _∎
    : (x : A)
    → x == x

  _∎ = λ x → idp

  -- Begin
  infix 1 begin_
  begin_
    : {x y : A}
    → x == y
    → x == y

  begin_ p = p

open EquationalReasoning public
