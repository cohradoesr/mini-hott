Empty type

A datatype without constructors is the empty type. This represents the falsehood.

data ⊥ {ℓᵢ} : Type ℓᵢ where

Synonyms of ⊥:

Empty = ⊥
𝟘     = ⊥

Its eliminator principle also called Ex falso quodlibet:

exfalso
  : ∀ {ℓ ℓᵢ} {A : Type ℓ}
  → ⊥ {ℓᵢ}
  --------
  → A

exfalso ()

Synonyms of exfalso rule:

Empty-elim = exfalso
⊥-elim     = exfalso
𝟘-elim     = exfalso

The negation function:

¬ : ∀ {ℓ} → Type ℓ → Type ℓ
¬ A = (A → ⊥ {lzero})

Unit type

The unit type is defined as a record to get also the $η$-rule definitionally. This type has no elimination rule.

record ⊤ {ℓ} : Type ℓ where
  constructor ★

{-# BUILTIN UNIT ⊤ #-}

Synonyms for the Unit type:

Unit = ⊤
𝟙    = ⊤

Synonyms for the data constructor:

unit : ∀ {ℓ} → ⊤ {ℓ}
unit = ★

Σ-type

We define Sigma types as a particular case of records in Agda.

infixr 60 _,_
record Σ {ℓᵢ ℓⱼ} (A : Type ℓᵢ)(C : A → Type ℓⱼ) : Type (ℓᵢ ⊔ ℓⱼ) where
  constructor _,_
  field
    π₁ : A
    π₂ : C π₁

open Σ public

Synonyms for its data constructors:

proj₁ = π₁
proj₂ = π₂

pr₁   = π₁
pr₂   = π₂

fst   = π₁
snd   = π₂

Π-types

Shorter notation for Π-types.

Π
  : ∀ {ℓᵢ ℓⱼ}
  → (A : Type ℓᵢ) (P : A → Type ℓⱼ)
  --------------------------------
  → Type (ℓᵢ ⊔ ℓⱼ)

Π A P = (x : A) → P x

Product type

Product type as a particular case of the Sigma type.

_×_
  : ∀ {ℓᵢ ℓⱼ}
  → (A : Type ℓᵢ) (B : Type ℓⱼ)
  ----------------------------
  → Type (ℓᵢ ⊔ ℓⱼ)

A × B = Σ A (λ _ → B)

Coproduct type

Sum types as inductive types

infixr 80 _+_
data _+_ {ℓᵢ ℓⱼ} (A : Type ℓᵢ) (B : Type ℓⱼ) : Type (ℓᵢ ⊔ ℓⱼ) where
  inl : A → A + B
  inr : B → A + B

Implication type

-- Implication.
data _⇒_ {ℓ}(A B : Type ℓ) : Type ℓ where
  fun : (A → B) → A ⇒ B

Biconditional type

-- Biconditional.
_⇔_ : ∀ {ℓ₁ ℓ₂} → Type ℓ₁ → Type ℓ₂ → Type (ℓ₁ ⊔ ℓ₂)
A ⇔ B = (A → B) × (B → A)

Boolean type

Boolean type, two constants true and false

data Bool : Type₀ where
  true  : Bool
  false : Bool

Synonyms:

𝟚  = Bool
𝟘₂ = false
𝟙₂ = true

We find many times writing the negation function, then let’s make it available:

neg¬ : Bool → Bool
neg¬ true  = false
neg¬ false = true

Booleans can be also defined using the Coproduct.

Natural numbers type

Natural numbers are the initial algebra for a constant and a successor function. The BUILTIN declaration allows us to use natural numbers in Arabic notation.

data ℕ : Type₀ where
  zero : ℕ
  succ : ℕ → ℕ

{-# BUILTIN NATURAL ℕ #-}

-- synonyms for natural numbers
Nat = ℕ
