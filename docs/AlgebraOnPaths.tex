Now, we can define a convenient syntax sugar for `ap` in
equational reasoning.

<pre class="Agda">
<a id="725" class="Keyword">infixl</a> <a id="732" class="Number">40</a> <a id="735" href="AlgebraOnPaths.html#389" class="Function">ap</a>

</pre>

Let's suppose we have a lemma:

```agda
  lemma : a ≡ b
  lemma = _
```

used in an equational reasoning like:

```agda
  t : a ≡ e
  t = f a ≡⟨ ap f lemma ⟩
      f b
      ∎
```


Then, we can now put the lemma in front:

```agda
  t : a == e
  t = f a =⟨ lemma |in-ctx f ⟩
      f b
      ∎
```
