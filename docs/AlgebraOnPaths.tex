Let's suppose we have a lemma:

```agda
  lemma : a ≡ b
  lemma = _
```

used in an equational reasoning like:

```agda
  t : a ≡ e
  t = f a ≡⟨ ap f lemma ⟩
      f b
      ∎
```


Then, we can now put the lemma in front:

```agda
  t : a == e
```
