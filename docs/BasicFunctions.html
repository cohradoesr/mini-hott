Basic functions

Identity functions

The identity function with implicit type.

id
  : {A : Type ℓ}
  ----------------
  → A → A

id = λ x → x

identity = id

The identity function on a type A is idf A.

idf
  : (A : Type ℓᵢ)
  ---------------
  → (A → A)

idf A = λ x → x

Constant functions

Constant function at some point b is cst b

cst
  : {A : Type ℓᵢ} {B : Type ℓⱼ}
  → (b : B)
  ---------
  → (A → B)

cst b = λ _ → b

constant = cst

Logical negation

neg¬
  : Bool
  → Bool

neg¬ tt = ff
neg¬ ff = tt

Composition

A more sophisticated composition function that can handle dependent functions.

_∘_
  : {A : Type ℓᵢ} {B : A → Type ℓⱼ} {C : (a : A) → (B a → Type ℓₖ)}
  → (g : {a : A} → ∏ (B a) (C a))
  → (f : ∏ A B)
  -------------------------------
  → ∏ A (λ a → C a (f a))

g ∘ f = λ x → g (f x)

infixr 80 _∘_

Synonym for composition (diagrammatic version)

_:>_
  : {A : Type ℓᵢ} {B : A → Type ℓⱼ} {C : (a : A) → (B a → Type ℓₖ)}
  → (f : Π A B)
  → (g : {a : A} → Π (B a) (C a))
  -------------------------------
  → Π A (λ a → C a (f a))

f :> g = g ∘ f

infixr 90 _:>_

Associativity of composition

  • Left associativity
∘-lassoc
  : {A B C D : Type ℓ}
  → (h : C → D) → (g : B → C) → (f : A → B)
  -----------------------------------------
  → (h ∘ (g ∘ f)) == ((h ∘ g) ∘ f)

∘-lassoc h g f = idp {a = (λ x → h (g (f x)))}
  • Right associativity
∘-rassoc
  : {A B C D : Type ℓ}
  → (h : C → D) → (g : B → C) → (f : A → B)
  -----------------------------------------
  → ((h ∘ g) ∘ f) == (h ∘ (g ∘ f))

∘-rassoc h g f = sym (∘-lassoc h g f)

Application

_$_
  : {A : Type ℓᵢ} {B : A → Type ℓⱼ}
  → (∀ x → B x)
  -------------
  → (∀ x → B x)

f $ x = f x

infixr 0 _$_

Coproduct manipulation

Functions handy to manipulate coproducts:

+-map
  : ∀ {i j k l} {A : Type i} {B : Type j} {A' : Type k} {B' : Type l}
  → (A → A')
  → (B → B')
  → A + B → A' + B'

+-map f g = cases (f :> left) (g :> right)

syntax +-map f g = 〈 f ⊕ g 〉

parallell
  : {A : Type ℓᵢ} {B : A → Type ℓⱼ} {C : (a : A) → B a → Type ℓₖ}
  → (f : (a : A) → B a)
  → ((a : A) → C a (f a))
  -------------------------
  → (a : A) → ∑ (B a) (C a)

parallell f g a = (f a , g a)

syntax parallell f g = 〈 f × g 〉
\

Curryfication

curry
  : {A : Type ℓᵢ} {B : A → Type ℓⱼ} {C : Σ A B → Type ℓₖ}
  → ((s : ∑ A B) → C s)
  -------------------------------
  → ((x : A)(y : B x) → C (x , y))

curry f x y = f (x , y)

Uncurryfication

unCurry
  : {A : Type ℓᵢ}{B : A → Type ℓⱼ}{C : Type ℓₖ}
  → (D : (a : A) → B a → C)
  ------------------------
  → (p : ∑ A B) → C

unCurry D p = D (proj₁ p) (proj₂ p)

uncurry
  : {A : Type ℓᵢ} {B : A → Type ℓⱼ} {C : (a : A) → B a → Type ℓₖ}
  → (f : (a : A) (b : B a) → C a b)
  ---------------------------------
  → (p : ∑ A B) → C (π₁ p) (π₂ p)

uncurry f (x , y) = f x y

Heterogeneous equality

data
  HEq (A : Type ℓ)
    : (B : Type ℓ)
    → (α : A == B) (a : A) (b : B)
    → Type (lsuc ℓ)
  where
  idp : {a : A} → HEq A A idp a a

Path functions

Composition of paths

_·_
  : {A : Type ℓ} {x y z : A}
  → (p : x == y)
  → (q : y == z)
  --------------
  → x == z

_·_ idp q = q

infixl 50 _·_

Inverse of paths

inv
  : {A : Type ℓ} {a b : A}
  → a == b
  --------
  → b == a

inv idp = idp

Synonyms for inverse path

_⁻¹ = inv
!_  = inv

infixl 60 _⁻¹ !_

Equational reasoning

Equational reasoning is a way to write readable chains of equalities like in the following proof.

  t : a ≡ e
  t =
    begin
      a ≡⟨ p ⟩
      b ≡⟨ q ⟩
      c ≡⟨ r ⟩
      d ≡⟨ s ⟩
      e
    ∎

where p is a path from a to b, q is a path from b to c, and so on.

module
  EquationalReasoning {A : Type ℓ}
  where

Definitional equals:

  _==⟨⟩_
    : ∀ (x {y} : A)
    -----------------
    → x == y → x == y

  _ ==⟨⟩ p = p

Synonyms:

  _==⟨idp⟩_  = _==⟨⟩_
  _==⟨refl⟩_ = _==⟨⟩_
  _≡⟨⟩_      = _==⟨⟩_

  infixr 2 _==⟨⟩_ _==⟨idp⟩_ _==⟨refl⟩_ _≡⟨⟩_

Chain:

  _==⟨_⟩_
    : (x : A) {y z : A}
    → x == y
    → y == z
    --------
    → x == z

  _ ==⟨ thm ⟩ q = thm · q

Synomyms:

  _≡⟨_⟩_ = _==⟨_⟩_

  infixr 2 _==⟨_⟩_ _≡⟨_⟩_

Q.E.D:

  _∎
    : (x : A)
    → x == x

  _∎ = λ x → idp

  infix 3 _∎

The begining of a proof:

  begin_
    : {x y : A}
    → x == y
    → x == y

  begin_ p = p

  infix 1 begin_

open EquationalReasoning public
