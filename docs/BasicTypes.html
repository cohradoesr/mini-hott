Equalities

In HoTT, we have a different interpretation of type theory in which the set-theoretical notion of sets for types is replaced by the topological notion of spaces.

The (homogeneous) equality type also called identity type is considered a primary type (included in the theory by default). To form this type, we fix a type A and a term a : A, to have the identity type a == a, also denoted by Id(a,a) or a‚áùa. We only have one constructor of these types called idp or sometimes refl. To use identity types, we have below the J-eliminator.
