Propositional truncation

Propositional truncation (or reflection) is the universal solution to the problem of mapping $X$ to a proposition $P$:

X  → ∥ X ∥
 ↘    ⇣ (!∃ h)
    ↘ P

Notes:

  • It’s possible to extend MLTT to get truncations for all types. (Such as resizing + funext, or higher inductive types.)

For a different way of formalising trucation see: agd-premises.

module
  TruncationType
  where

  private
    data
      !∥_∥ {ℓ} (A : Type ℓ)
        : Type ℓ
        where
        !∣_∣ : A → !∥ A ∥

  ∥_∥
    :  (A : Type ℓ)
    → Type ℓ

  ∥ A ∥ = !∥ A ∥

  prop-trunc = ∥_∥ 

  ∣_∣
    : {X : Type ℓ}
    → X
    → ∥ X ∥

  ∣ x ∣ = !∣ x ∣

  ∥∥-intro = ∣_∣

Any two elements of the truncated type are equal

  postulate
    trunc
      : {A : Type ℓ}
      → isProp ∥ A ∥

Recursion principle

  trunc-rec
    : {A : Type ℓᵢ} {P : Type ℓⱼ}
    → isProp P
    → (A → P)
    ---------
    → ∥ A ∥ → P

  trunc-rec _ f !∣ x ∣ = f x

  trunc-elim = trunc-rec
  ∥∥-rec     = trunc-rec

There exists the possibility to charactherize, propositional truncation using an impredicative approach, which means, our definition will lay on a larger universe as follows on the right-hand side.

Remarks:

  • rhs is a propositon assuming funext
  • this equation tells us an important relation (a pattern) between the type (in this case, prop-trunc) and its elimination principle (trunc-rec)
  • Impredicative means in this context: “it is defined in terms of quantification over a family which the thing we are defining is a member of.” (Gylterud).

Truncation Lemmas

  truncated-is-prop
    : {A : Type ℓ}
    → isProp (∥ A ∥)
  truncated-is-prop = trunc

  ∥∥-is-a-prop = truncated-is-prop

  prop-≃-truncated
    : { A : Type ℓ}
    → isProp A
    → ∥ A ∥ ≃ A

  prop-≃-truncated pA = lemma333 trunc pA (trunc-rec pA id) ∣_∣

  trunc-≃ = prop-≃-truncated

A relation between double implication and the truncation of a type:

  postulate
   trunc-⇔-¬¬
     : ∀ {ℓ} {X : Type ℓ}
     → ∥ X ∥ ⇔ (¬ (¬ X))
