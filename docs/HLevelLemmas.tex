### Proposition and Set lemmas

<pre class="Agda">
<a id="690" class="Keyword">module</a> <a id="HLevelLemmas"></a><a id="697" href="HLevelLemmas.html#697" class="Module">HLevelLemmas</a> <a id="710" class="Keyword">where</a>

{: .equation }
  $$ \isContr{A} ⇒ \isProp{A} ⇒ \isSet{A}.$$
